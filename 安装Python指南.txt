========================================
  Python 安装指南（新手必看）
========================================

你的电脑还没有安装 Python，跟着下面步骤操作即可。


⚠️ 重要警告 ⚠️

不要用 Microsoft Store 安装 Python！Store 版功能残缺，
脚本跑不起来。必须用下面官网地址下载！


--------------------------------------------------
第一步：下载 Python（用 Edge 浏览器打开）
--------------------------------------------------

官网下载地址：
https://www.python.org/downloads/

页面会自动推荐最新版本，点黄色大按钮下载
（文件名类似 python-3.12.x-amd64.exe）


--------------------------------------------------
第二步：安装 Python
--------------------------------------------------

1. 双击刚刚下载的 python-3.xx.x-amd64.exe

2. ⚠️ 最重要的一步 ⚠️
   安装界面底部有一个复选框：
   ☐ Add Python to PATH
   一定要勾上！！！不勾后面什么都跑不了！！！

3. 勾上之后，点 "Install Now"


--------------------------------------------------
第三步：验证安装
--------------------------------------------------

1. 按 Win + R，输入 cmd，回车

2. 在黑色窗口输入：
   python --version

3. 显示 Python 3.12.x 说明安装成功


--------------------------------------------------
第四步：回到 ESurfingDialer-Pro
--------------------------------------------------

双击 setup.bat 继续配置校园网自动认证。
